Amigo Secreto - Sorteador de Nomes
Este é um projeto simples onde o usuário pode adicionar nomes e sortear um deles aleatoriamente. O projeto foi desenvolvido utilizando HTML, CSS e JavaScript.

Como Funciona:
Adicionar Nomes: O usuário pode digitar um nome no campo de texto e clicar no botão "Adicionar" para incluir o nome na lista.

Lista de Nomes: Os nomes adicionados são exibidos em uma lista abaixo do campo de entrada.

Sortear Nome: Após adicionar os nomes, o usuário pode clicar no botão "Sortear" para sortear aleatoriamente um nome da lista. O nome sorteado será exibido na tela em verde.

Estrutura do Projeto:
HTML: A estrutura da página é definida no arquivo index.html. Ele contém o formulário para adicionar nomes, a lista de nomes e o botão para realizar o sorteio.

CSS: O estilo da página é definido no arquivo style.css. Ele inclui estilos para o layout, cores, fontes e botões.

JavaScript: A lógica para adicionar nomes e realizar o sorteio é realizada no script embutido no arquivo HTML.

Onde Usar:
Foi feito o deploy do projeto pelo Vercel. Você pode acessar pelo link ao lado: https://challenger-sorteador.vercel.app/.

Personalização
Estilos: Você pode personalizar o estilo da página editando o arquivo style.css.

Funcionalidades: Adicione novas funcionalidades ao script JavaScript, como a possibilidade de remover nomes da lista ou sortear mais de um nome.
